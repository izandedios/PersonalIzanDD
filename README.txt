Descripción del Proyecto
Este proyecto es una página web personal desarrollada por Izan De Dios, con el objetivo de darme a conocer laboralmente. La página muestra una presentación personal, los proyectos en los que he trabajado y un formulario de contacto.

Tecnologías Utilizadas
HTML
CSS
Bootstrap
jQuery
Google Fonts
GitHub Pages
JavaScript

La página está dividida en las siguientes secciones:

1. Página principal:
Contiene una carta de visita que describe quién soy, mi formación, y mis intereses personales.
Es la introducción a la web y da la bienvenida a los visitantes.
2. Proyectos:
Sección en formato CARDS que expone algunos de mis proyectos más relevantes.
Al hacer clic en una tarjeta de proyecto, se despliega información adicional usando jQuery para mostrar secciones con más detalles.
Además, incluye un buscador que permite filtrar los proyectos utilizando la función jQuery.contains().
3. Contacto:
Un formulario con validación que permite a los usuarios ponerse en contacto conmigo.
Si la validación es exitosa (como la verificación del correo electrónico), se muestra un modal de confirmación de envío.
4. Footer:
Incluye mi nombre y los derechos de autor.
Este footer está presente en todas las secciones de la página y proporciona coherencia en el diseño.
Funcionalidades Destacadas
SPA (Single Page Application): Toda la navegación se realiza en una única página HTML, utilizando jQuery para mostrar y ocultar secciones según las interacciones del usuario.
Buscador: Implementado con jQuery, permite a los usuarios buscar proyectos filtrando el contenido de las tarjetas.
Interacción dinámica con modales: Los detalles adicionales de los proyectos se muestran mediante el uso de jQuery para hacer aparecer contenido dinámicamente.
Validación de formulario de contacto: Se asegura que el usuario introduzca un correo válido antes de enviar el formulario.
